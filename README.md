# VAEABMIL

This repository contains the original implementation of the VAEABMIL model as described in the paper [Using Variational Autoencoders for Out of Distribution Detection in Histological Multiple Instance Learning](https://ieeexplore.ieee.org/document/11098836/).


![Pipeline](assets/pipeline.jpg)

However, for the most up-to-date and recommended version, please use the implementation available in the [torchmil](https://github.com/user/torchmil) library under `torchmil.models.VAEABMIL`. The torchmil version includes additional features such as support for multi-output classification and further enhancements.
